# Facenet-Based Face Recognition App

This project is a face recognition application built using **Facenet-PyTorch**. It allows users to upload an image, detect faces, and match them against a pre-trained dataset using a simple web-based UI powered by **Streamlit**.

---

## Features

- **Upload Images**: Upload images for face detection and recognition.
- **Interactive UI**: User-friendly web interface using Streamlit.
- **Face Matching**: Matches uploaded faces against a pre-trained dataset.

---

## Requirements

- Python 3.8 or later
- Pre-trained dataset (`data.pt` file)
